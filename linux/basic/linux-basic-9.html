                                                                                                      # 41 安装 gcc

                                                                                                      有网情况：yum install gcc

                                                                                                      yum install gcc-c++

                                                                                                      无网情况下：

                                                                                                      gcc 相关 rpm 文件下载地址

                                                                                                      链接：https://pan.baidu.com/s/1UHMVEOZ4gTRVlZBqHcVqqQopen in new window

                                                                                                      提取码：41ot

                                                                                                      进入 CentOS 下存放 .rpm 文件的目录，执行

                                                                                                      rpm -Uvh *.rpm --nodeps --force

                                                                                                      然后使用

                                                                                                      gcc -v

g++ -v

                                                                                                      查看 gcc 版本和 g++ 版本，会看到详细的版本信息，离线环境下安装 GCC 和 GCC-C++ 就完成了
